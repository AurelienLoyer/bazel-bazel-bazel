Aujourd’hui nous allons parler de Build System. Pour mettre tout le monde au clair, qu’est-ce que c’est un build system. C’est une toolchain qui à partir d’inputs va executer certaines commandes les unes à la suite des autres afin de créer vos livrables. Durant cette toolchain, nous n’avons bien evidemment pas que la tache de creation du livrable, nous avons en plus de cela pdes taches permettant de tester notre code, vérifier la qualité, …​
